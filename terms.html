2. Description of Service

Inbox Sweeper is a Chrome extension designed to help users manage their Gmail inbox by:

  • Extract unique email IDs from your inbox.
  • Delete emails from selected senders based on user action.
  • Export email data as CSV files.

This extension requires access to your Gmail inbox through Google's OAuth 2.0 authentication and only operates when initiated by the user.

8. WhatsApp PDF Delivery Feature

Inbox Sweeper offers an optional automation feature that allows users to receive PDF versions of their latest Gmail emails directly to their WhatsApp number via the WhatsApp Cloud API.

8.1 Feature Overview

  • ✅ The extension captures incoming Gmail messages in print format and converts them into PDFs using html2canvas and jsPDF.
  • ✅ These PDFs are then sent to the user’s verified WhatsApp number via an API integration.
  • ✅ The user must explicitly enable this feature from the extension UI and complete OTP-based mobile number verification.

8.2 User Responsibilities

  • 📲 Users are responsible for providing a valid WhatsApp number for verification via OTP.
  • 🔄 Users must ensure they are compliant with WhatsApp’s terms and conditions before using this feature.
  • 📤 Users acknowledge that email content is converted and sent only upon receiving new emails and only to their verified number.

8.3 Privacy & Data Handling

  • 🔐 No email content, PDF, or WhatsApp number is stored on external servers.
  • 🔒 WhatsApp API tokens are securely fetched and never exposed or stored in browser storage.
  • 🚫 We do not log, analyze, or share any PDFs generated or sent via WhatsApp.

Important: Inbox Sweeper does not store, monitor, or retain any WhatsApp messages, phone numbers, or generated PDFs beyond the delivery process.
