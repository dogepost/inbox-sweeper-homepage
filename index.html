Inbox Sweeper - Chrome Extension

Organize, clean, and automate Gmail with one click.

🌟 Get the Extension

Why Use Inbox Sweeper?

Advanced automation to keep Gmail clean and clutter-free.

  • Extract unique email IDs
  • Bulk delete by email ID
  • Export email IDs to CSV
  • Filter emails via checkbox
  • Show unsubscribe links
  • Convert new emails to PDF and send to WhatsApp
  • Set a reply deadline while composing emails and track responses automatically

Why Does It Need Gmail Access?

Secure OAuth 2.0 used to fetch your Gmail data locally for:

  • Email analysis & filtering
  • Bulk email actions
  • Unsubscribe link extraction

No data stored. Everything happens locally.

✨ Core Feature

Automated Gmail to WhatsApp PDF (Language Support: English)

Monitors your inbox and converts emails to PDFs sent directly to your verified WhatsApp using the WhatsApp Cloud API.

🔒 Secure Verification

OTP-Based Mobile Verification

Verifies your WhatsApp number before PDF delivery using a 4-digit OTP via WhatsApp itself.

📑 Smart PDF Generator

PDF Formatting

  • Uses html2canvas and jsPDF
  • Captures sender, subject, date, and attachments
  • Mobile-friendly A4 layout
  • Clean footer branding
⚙️ Customization

Filters and Scheduling

  • Search emails instantly by sender or email ID
  • Filter emails by Gmail categories (Primary, Updates, Social, Promotions)
  • Filter by sender or subject for precise control
  • Set auto-delivery intervals
  • Enable/disable WhatsApp delivery
📂 Data Export

Export Email IDs to CSV

  • Extracts all unique sender email IDs from your Gmail inbox
  • Downloads data instantly in CSV format (Excel/Google Sheets ready)
  • Clean, structured list for easy sorting and analysis
  • Useful for backups, marketing, and contact management
🔐 Privacy First

Secure API Integration

  • Encrypted WhatsApp token storage
  • No third-party data sharing
  • Direct communication with Cloud API
🌐 Cloudflare Protected

Secure Token Proxy

Inbox Sweeper fetches tokens securely via Cloudflare Worker to avoid client-side token exposure.

🔔 Real-Time Triggers

Background Monitoring

Uses Chrome alarms API to watch Gmail with minimal performance impact. Stops auto-delivery when not verified.

⏰ Smart Follow-ups

Reply by Date & Time

Set a reply deadline while composing emails and track responses automatically. Inbox Sweeper highlights emails with smart badges: Upcoming Today Overdue — so you never miss important replies.

🛡 Smart Protection

Smart Delete Filters (Safe Bulk Cleaning)

Clean your inbox confidently with intelligent delete controls. Before removing emails, you can exclude important ones like ⭐ Starred, ❗ Important, 📩 Unread, and 📎 Attachments — ensuring no critical data is lost during bulk deletion.

🛠️ Button Functionality

  • 📥 Extract Unique Emails: Scan your inbox and collect all unique sender email IDs instantly.
  • 🗑 Bulk Delete: Delete all emails from selected senders in one click with smart safety filters.
  • 📂 Export CSV: Download all extracted email addresses as a CSV file for backup or analysis.
  • 🔍 Search & Filter: Quickly find emails by sender or filter by Gmail categories like Primary, Social, Promotions, and Updates.
  • ⚪ Select Email: Choose specific email IDs to perform targeted actions like delete or unsubscribe.
  • 📤 Unsubscribe Links: Automatically detect and display unsubscribe links from emails.
  • 📅 Reply Deadline: Set “Reply by Date & Time” and track follow-ups with visual indicators.
  • 🛡 Smart Delete Filters: Exclude starred, important, unread, or emails with attachments before deleting.
  • 📜 View Logs: Keep track of all actions performed for transparency.
  • 🧹 Clear Logs: Instantly erase all stored activity logs.
  • 📥 Download Logs: Export logs as a CSV file for reporting or backup.
